## 后台管理系统学习样例

> 本项目是跟着尚硅谷`vue3`实战项目硅谷甄选学习完成

> 但是只制作了第一个页面的布局和功能，手写了第一个页面的后端和数据库

> 需要配合本人仓库对应的后端项目使用，否则页面无法打开

## 使用方法

- 克隆项目

```bash
git clone https://github.com/jgfzy/vue3_admin.git
```

- 进入项目目录后

```bash
pnpm i
```

- 运行项目(需要配套后端服务器开启)

```bash
pnpm run dev
```
